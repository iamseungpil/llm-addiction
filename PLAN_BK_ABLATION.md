# BK 표상 인과성 검증 실험 계획서

> **목표**: LLM 내부의 파산(BK) 표상이 모델의 배팅 결정에 인과적 역할을 하는지 검증한다.
>
> **핵심 질문**: BK 표상을 조작하면 모델의 파산 행동이 바뀌는가?

---

## 1. 배경

V10 보고서는 Gemma-2-9B-IT와 LLaMA-3.1-8B-Instruct 내부에 파산을 예측하는 뉴런이 존재함을 보였다 (분류 AUC > 0.95, 보편적 BK 뉴런 Gemma 600개 / LLaMA 1,334개). **그러나 이 증거는 모두 상관적이다** (V10 Section 6.2).

| 알고 있는 것 | 모르는 것 |
|-----------|---------|
| 파산한 게임에서 특정 뉴런이 활성화된다 | 그 뉴런이 파산의 **원인**인지 **부산물**인지 |

이 실험이 성공하면 V10의 상관적 발견이 인과적 메커니즘으로 격상된다.

---

## 2. 선행 실험 결과 (뉴런 단위 ablation)

LLaMA L22에서 |r| ≥ 0.2인 보편적 BK 뉴런 (promoting 104개 + inhibiting 89개)에 대해 zero ablation과 mean patching을 수행한 결과, **모든 조건에서 효과가 유의하지 않았다** (50회 SM 게임, Fisher's exact test p > 0.5).

| 조건 | stop rate 변화 | p-value |
|------|-------------|---------|
| Promoting zero ablation | +0.04 | 0.84 |
| Inhibiting zero ablation | +0.06 | 0.69 |
| Promoting mean patching | +0.08 | 0.55 |
| Random (통제) | -0.04 | 0.84 |

**결론: 개별 뉴런 단위 조작으로는 BK 행동을 바꿀 수 없다.**

이는 BK 정보가 개별 뉴런이 아닌 **분산 표상(distributed representation)**에 인코딩됨을 시사한다. V10의 "은닉 상태 > SAE 전이" 결과와 일치한다.

**단, 이것이 BK 표상이 부산물이라는 뜻은 아니다.** 개별 뉴런이 아닌 활성화 공간의 **방향(direction)** 수준에서 조작해야 분산 표상의 인과 역할을 검증할 수 있다.

---

## 3. 수정된 실험 설계

선행 결과를 반영하여, **뉴런 단위 ablation은 제외**하고 **방향 수준 조작**에 집중한다.

### Phase 1: Steering (BK 방향 벡터 덧셈) — 핵심 실험

**소요: 구현 반나절, 실행 반나절**

#### 선행 실험과 무엇이 다른가

| | 선행 실험 (뉴런 ablation) | 이 실험 (Steering) |
|---|---|---|
| 조작 단위 | 개별 뉴런 104-189개 | 활성화 공간의 방향 벡터 (4,096D 전체) |
| 조작 방식 | 특정 뉴런을 끄거나 값 대체 | 전체 활성화에 방향 벡터를 더하거나 빼기 |
| 분산 표상에 적합한가 | 아니오 — 개별 뉴런은 분산 표상의 일부만 건드림 | **예** — 방향 벡터는 분산된 정보를 한꺼번에 조작 |

#### 방법

V10이 이미 BK 방향 벡터를 계산해놨다 (Section 4.3: BK 게임 평균 - Safe 게임 평균).

```python
# forward hook (L22)
def steering_hook(module, input, output):
    output += alpha * bk_direction_vector
    return output
```

#### 조건

| 조건 | alpha | 예상 |
|------|-------|------|
| +BK (약) | +0.5 | BK 비율 소폭 증가 |
| +BK (중) | +1.0 | BK 비율 증가 |
| +BK (강) | +2.0 | BK 비율 대폭 증가 |
| -BK (약) | -0.5 | BK 비율 소폭 감소 |
| -BK (중) | -1.0 | BK 비율 감소 |
| -BK (강) | -2.0 | BK 비율 대폭 감소 |
| Baseline | 0 | 기준선 |
| Random direction | ±1.0 (무작위 방향) | 변화 없음 (통제) |

조건당 IC 400게임. 총 3,200게임.

#### 성공 기준

- alpha에 비례하여 BK 비율이 단조 변화 (dose-response) → **분산 표상 수준의 인과 증거**
- +alpha에서 BK↑, -alpha에서 BK↓ → 양방향 효과
- Random direction에서는 변화 없음 → BK 방향에 특이적

#### 실패 시

- 모델 출력이 깨진다 → alpha를 줄여 재시도 (0.1, 0.2, ...)
- BK 비율 변화 없음 → Phase 2로 진행

---

### Phase 2: 다층 Steering

**소요: Phase 1 코드 재활용, 실행 1일**

Phase 1이 L22 단일 층에서 효과가 없을 경우, BK 정보가 여러 층에 분산되어 있을 수 있다.

#### 방법

Phase 1과 동일하되, 여러 층에 동시에 steering을 적용한다.

| 조건 | 적용 층 | 근거 |
|------|--------|------|
| L22 only | L22 | Phase 1과 동일 (기준) |
| Deep layers | L22 + L25 + L30 | V10에서 BK 분류 성능이 높은 층 |
| All checkpoints | L8 + L12 + L22 + L25 + L30 | LLaMA 전체 분석 층 |

alpha = Phase 1에서 가장 효과적이었던 값 사용.

#### 성공 기준

- 다층 steering에서 BK 비율 변화가 단일 층보다 큼 → BK 정보가 다층 분산
- 특정 층 조합에서만 효과 → 핵심 층 식별

---

### Phase 3: 게임 규모 확대 재검증

**소요: 실행 1일**

#### 왜 필요한가

선행 실험의 약점 중 하나: **50회 게임은 통계적 검정력이 매우 낮다.** Fisher's exact test에서 50회로는 효과 크기가 상당히 커야만 유의하게 나온다. 작은 효과가 있었더라도 놓쳤을 수 있다.

#### 방법

Phase 1-2에서 가장 유망한 조건을 선택하여, **게임 수를 400-800으로 확대**하여 재실행한다.

| | 선행 실험 | 이 실험 |
|---|---|---|
| 게임 수 | 50 | 400-800 |
| 효과 크기 0.10 탐지 가능? | 아니오 (검정력 ~15%) | 예 (검정력 ~80%) |

---

## 4. 전체 실행 흐름

```
Phase 1 (L22 Steering)  ─── 효과 있음 → Phase 3 (규모 확대 재검증)
       │
       └── 효과 없음 ──→ Phase 2 (다층 Steering)  ─── 효과 있음 → Phase 3
                               │
                               └── 효과 없음 → "BK 표상은 인과적이지 않음"으로 보고
```

---

## 5. 기대 효과

| 시나리오 | 의미 | 논문 영향 |
|---------|------|---------|
| Phase 1에서 dose-response 성공 | BK 방향이 행동을 직접 제어 | 최강 인과 증거, V10 격상 |
| Phase 2에서 다층 steering 성공 | BK가 다층 분산 표상으로 인과적 | 인과 증거 + 분산 메커니즘 발견 |
| 전부 실패 | BK 표상은 정보적이지만 인과적이지 않음 | 의미 있는 부정적 결과 (선행 뉴런 ablation과 합쳐 "표상 ≠ 인과" 논증) |
