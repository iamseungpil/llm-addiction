# V12 실험 계획서: 무작위 방향 통제 및 확장

> V11의 steering 실험에서 BK 방향의 dose-response (ρ=0.927, p=0.003)가 확인되었으나,
> 무작위 방향 α=1에서도 BK 방향 α=-2와 유사한 행동 변화가 관찰되었다.
>
> **핵심 질문: dose-response가 BK 방향에 특이적인가, 아니면 아무 방향이든 나타나는가?**

---

## 1. 왜 이 실험이 필요한가

V11 Table 12 결과:

| 조건 | BK 율 |
|------|------|
| BK 방향 α=-2 | 0.340 |
| BK 방향 α=0 (기준) | 0.480 |
| BK 방향 α=+2 | 0.520 |
| **무작위 방향 α=1** | **0.340** |

무작위 방향 α=1의 BK 율(0.340)이 BK 방향 α=-2(0.340)와 동일하다. 두 가지 해석이 가능하다:

| 해석 | 의미 |
|------|------|
| A: BK 방향이 특별하다 | 무작위 방향은 "혼란"으로 보수적 행동 유발, BK 방향만이 방향성 있는 dose-response를 생성 |
| B: 아무 방향이든 비슷하다 | 잔차 스트림을 어디로든 밀면 행동이 바뀜, BK 방향이 특별하지 않음 |

A와 B를 구분하는 유일한 방법: **무작위 방향에서도 α를 변화시켜 dose-response가 나오는지 확인**

---

## 2. 실험 설계

### 실험 1: 무작위 방향 다중-α (최우선)

V11에서 무작위 방향은 α=1 하나만 테스트했다. 이것을 BK 방향과 동일한 α 범위로 확장한다.

| α | BK 방향 (V11 완료) | 무작위 방향 (새로 실행) |
|---|----------------|----------------|
| -2.0 | 0.340 | ? |
| -1.0 | 0.480 | ? |
| -0.5 | 0.420 | ? |
| 0 | 0.480 | 0.480 (동일) |
| +0.5 | 0.500 | ? |
| +1.0 | 0.520 | 0.340 (V11) |
| +2.0 | 0.520 | ? |

무작위 방향 5개 이상을 샘플링하여, 각 α에서 평균 BK 율과 표준편차를 계산한다.

**소요: 구현 1시간 (V11 코드 재활용), 실행 반나절**

#### 판단 기준

```
무작위 방향에서:
  - dose-response 없음 (α와 BK 율 무관) → BK 방향 특이적 (해석 A 확인)
  - dose-response 있음 (α에 비례하여 BK 율 변화) → 비특이적 (해석 B)
  - 일부 방향에서만 dose-response → 방향 공간의 구조에 의존 (추가 분석 필요)
```

### 실험 2: 다층 조향 (Phase 2)

V11은 L22 단일 층에서만 실행했다. L22 + L25 + L30에 동시 steering을 적용하면 효과가 증폭되는지 확인한다.

| 조건 | 적용 층 |
|------|--------|
| L22 only | L22 (V11과 동일, 기준) |
| Deep layers | L22 + L25 + L30 |
| All checkpoints | L8 + L12 + L22 + L25 + L30 |

α = V11에서 가장 효과적이었던 +2.0 / -2.0 사용. 조건당 50게임.

**소요: 구현 30분, 실행 반나절**

### 실험 3: Gemma 교차 검증

V11의 steering은 LLaMA SM에서만 수행되었다. Gemma에서도 동일 결과가 나오는지 확인하면 교차 모델 인과 증거가 된다.

**소요: 구현 30분 (모델 교체만), 실행 반나절**

### 실험 4: 규모 확대 (선택)

V11은 조건당 n=50이었다. 실험 1의 결과가 경계선(borderline)이면 n=200으로 확대하여 검정력을 높인다.

**소요: 실행 1일**

---

## 3. 실행 순서

```
실험 1 (무작위 다중-α)  ← 최우선, 1일
  │
  ├── BK 방향 특이적 확인 → 실험 2 (다층) + 실험 3 (Gemma) 병렬
  │                         └── 인과 증거 확립, V12 완성
  │
  └── 비특이적 → "steering은 비특이적 교란"으로 보고
                  └── 방향 조향 이외의 인과 검증 방법 탐색 필요
```

## 4. 기대 효과

| 시나리오 | V12 결론 |
|---------|--------|
| 실험 1에서 BK 방향만 dose-response | **"BK 표상은 도박 행동에 인과적 영향을 미치며, 이는 BK 방향에 특이적이다"** |
