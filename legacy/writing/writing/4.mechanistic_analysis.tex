\section{SAE Feature Activation Patterns Across All Layers}
\label{SAE-feature}

This appendix provides a comprehensive visualization of all features from layers 25 through 31 that exhibited a strong separating effect, categorized into positive (risky) and negative (safe) activations.

\begin{figure}[h!]
    \centering
    \includegraphics[width=0.9\textwidth]{iclr2026/images/appendix_centered_violin_plots.pdf} 
    \caption{Complete activation distributions for maximum separation features across all layers (25--31). This comprehensive analysis shows the seven features with minimal overlap between bankrupt and safe groups, selected from 6,400 LLaMA experiments. Features are arranged by layer with Cohen's $d$ values ranging from -1.300 to +1.669, all significant at $p < 0.001$. This complete set provides the full context for the representative features shown in Figure~\ref{fig:feature-separation} of the main text.}
    \label{fig:sae_complete_analysis}
\end{figure}
