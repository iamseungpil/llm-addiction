# Legacy Results

This directory stores historical or presentation-only artifacts that remain
useful for reference but are not current paper-critical sources.

## What belongs here

- historical monitor traces
- build companion artifacts such as `*.aux`, `*.log`, `*.out`, `*.toc`
- old integrated-study snapshots when they are no longer first-line references
- duplicate presentation exports

## What does not belong here

- current paper manifests
- canonical raw JSON used by the Korean paper
- active experiment logs
- any raw data file required by a runner

## Rule

If a file is still used by a current script, paper manifest, or active monitor,
it should stay in its original location.
