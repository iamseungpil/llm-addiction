## 2. Hypotheses

Each uses Plan v3's primary statistic: **CA-Z perm_p for binary (BK)**, **OLS-slope perm_p for continuous (I_BA, wealth)**. Confirmatory task per experiment has α=0.05 (no correction); exploratory tasks use Holm.

### H1 — Exp A LLaMA IC Phase 2 significant
- **Claim**: CA-Z perm_p < 0.05 OR OLS-I_BA perm_p < 0.05
- **Prior**: Phase 1 ckpt bk_rate span 0.08 → 0.50 (6x range). Strong dose-response expected.
- **Falsification**: both perm_p ≥ 0.1

### H2 — Exp A LLaMA MW Phase 2 significant
- **Claim**: CA-Z perm_p < 0.05
- **Prior**: Phase 1 ckpt bk_rate 0.675 (baseline 0.758). Range 0.33 → 0.95 likely.
- **Falsification**: CA-Z perm_p ≥ 0.1

### H3 — Exp B Gemma IC (confirmatory) significant
- **Claim**: CA-Z perm_p < 0.05 OR OLS-I_BA perm_p < 0.05
- **Prior**: Phase 1 ckpt bk_rate 0.105 (baseline 0.108). Highest Gemma BK rate.
- **Falsification**: both perm_p ≥ 0.1

### H4 — Shared-axis (E2-causal) — per-model falsifier
- **Strong**: LLaMA ≥2/3 tasks AND Gemma IC perm_p < 0.05 (rank-1 causal across models)
- **Medium**: LLaMA ≥2/3 OR Gemma IC sig (uni-model evidence)
- **Null**: LLaMA <2/3 AND Gemma IC not sig
- **Gemma SM/MW expected floor** — not part of falsifier due to baseline sparsity

### H5 — V14/V16 5-layer rerun replicates 2026-04-08 paper
- **Claim**: L8/L22/L25/L30 perm_p differences with paper's numbers < 0.2; rho within ±0.1
- **Falsification**: any layer's perm_p shifts by > 0.3 (points at implementation regression, not axis absence)

### (dropped) H3-v4 "Gemma SM/MW floor": downgraded to **descriptive sanity check** — report CA-Z + bk_rate range, no hypothesis test
