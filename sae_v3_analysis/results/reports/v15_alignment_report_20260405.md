**날짜:** 2026-04-05  
**목적:** 현재 논문 주장과 실험 증거의 정합성을 감사하고, 추가 실험 계획을 검증한다.

## 1. 발견된 문제

### 문제 1: "3--48배" 주장이 부정확 (심각도: 높음)

**현재 논문 표현:**
- Abstract: "파산율을 3--48배 증가시키고"
- Discussion: "가변 베팅은 파산율을 3--48× 증가시켰으며"

**실제 데이터:**

| 모델 | Fixed BK% | Variable BK% | 배수 |
|------|-----------|-------------|------|
| GPT-4o-mini | 0.00% | 21.31% | ∞ (기저=0) |
| GPT-4.1-mini | 0.00% | 6.31% | ∞ (기저=0) |
| Gemini-2.5 | 3.12% | 48.06% | **15.4×** |
| Claude-3.5 | 0.00% | 20.50% | ∞ (기저=0) |
| LLaMA-3.1 | 2.62% | 6.75% | **2.6×** |
| Gemma-2 | 12.81% | 29.06% | **2.3×** |

**문제:** "48배"는 존재하지 않음. 계산 가능한 범위는 2.3×--15.4×.
"48"은 파산율 퍼센트(48%)와 배수를 혼동한 것.

**수정안:** "파산율을 0--13%에서 6--48%로 증가시켰다" (비율 범위로 표현)

### 문제 2: within-bet-type AUC 범위 불일치 (심각도: 중간)

**현재 논문:** "within-bet-type AUC 0.885--0.995"  
**실제 Table:** Gemma within-bet-type R1 AUC = 0.617--0.805  

이 범위(0.885--0.995)는 LLaMA 결과로 추정되지만, 본문에서 어떤 모델의 수치인지 명시되지 않음.

**수정안:** 모델별로 구분하여 표기

### 문제 3: Temperature 실험의 프롬프트 차이 (심각도: 낮음)

Temperature 실험은 원래 행동 실험과 프롬프트 형식이 다름:
- 원래: ROLE_INSTRUCTION + "Explain your reasoning" + "Final Decision: Bet $X or Stop"
- Temperature 실험: "Option 1/2" 형식 (V12 steering 스타일)

이로 인해 Fixed BK rate이 원래 2.62%에서 20%로 상승.
**단, Temperature 실험의 목적은 within-experiment Fixed vs Variable 비교이므로, 절대적 BK rate 차이는 결론에 영향을 주지 않음.**

## 3. 추가 실험 현황 및 계획 검증

### 실험 A: Balance Confound 분석 — ✅ 완료, 논문 반영 완료

| 항목 | 내용 |
|------|------|
| **의도** | 신경 분류가 잔액 정보 읽기가 아님을 증명 |
| **가설** | 잔액 동일 구간에서도 AUC 유지 |
| **검증** | $95-105 매칭 → AUC 0.994 (baseline 0.971 초과) |
| **결론** | **가설 지지.** 분류 신호는 잔액 외 내부 표상에 기반 |
| **논문 주장과 정합** | ✅ RQ1 "파산 예측 패턴이 존재한다"를 강화 |

### 실험 B: Temperature 대조 — 🔄 진행 중 (12.5%)

| 항목 | 내용 |
|------|------|
| **의도** | Variable>Fixed 패턴이 sampling noise가 아닌 체계적 경향임을 증명 |
| **가설** | temp=0.0(결정적)에서도 Variable>Fixed 유지 |
| **중간 결과** | temp=0.0: Fixed 20% vs Variable 86--88% → **가설 강하게 지지** |
| **논문 주장과 정합** | ✅ "자율성이 비합리성을 증폭"의 강건성 보강 |

**주의사항:** 프롬프트가 원래 실험과 다르므로, 논문에서 "추가 temperature 대조 실험"으로 명시하고 within-experiment 비교에 초점. 원래 실험의 절대적 수치와 직접 비교하지 않음.

### 실험 C: V14 Permutation Test — 🔄 진행 중

| 항목 | 내용 |
|------|------|
| **의도** | 직접 개입의 방향 특이성 검증 |
| **가설** | BK 방향이 무작위 방향보다 유의하게 강한 steering 효과 |
| **현황** | Exp1 Random 10/10 진행 중, Exp4 Gemma 3/10 |
| **논문 주장과 정합** | ✅ 부록에 보수적으로 정리하는 구조와 일치 |
